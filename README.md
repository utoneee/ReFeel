# 能量補給站 ReFeel

朋友小圈子共享正能量的扭蛋機 app。把圖片、GIF、影片連結、語錄丟進去，隨機抽一個出來。

## 功能

- **三種扭蛋機台**
  - 混合：所有人的公開內容一起抽
  - 個人：自己上傳的公開內容 + 私密內容
  - 自訂：自選來源（公開/私密）和類型（語錄/圖片/GIF/影片）
- **身份系統**：選一個愛心代號（💚🧡💛🩶💙），不需要帳號
- **公開 / 私密**：公開內容對所有人可見，私密內容只存在自己的裝置
- **內容類型**：圖片、GIF、YouTube 影片連結、文字語錄
- **管理牆**：瀏覽、篩選、刪除所有扭蛋

## 使用方式

直接開啟 `index.html`，或部署到靜態主機後用同一個網址分享給朋友。

## 技術

- 單一 HTML 檔（CSS + JS 內嵌）
- 目前資料存於 localStorage
- 計畫：公開內容遷移至 Firebase Firestore + Storage，部署至 Firebase Hosting
